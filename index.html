Parágrafos e Formatação de Texto

O parágrafo padrão é definido pela tag <p>. Dentro de um parágrafo, você pode usar várias outras tags para formatar o texto:

Este texto contém uma palavra marcada, uma palavra em tamanho pequeno, uma palavra em itálico, uma palavra sublinhada, uma palavra em negrito, e uma palavra com riscado.

Você também pode usar HTML para definir abreviações.

A tag <font> pode ser usada para modificar a cor, o tamanho e a face da fonte. Por exemplo:

Este texto é vermelho, tamanho 4, e usa a fonte Verdana.

Listas Ordenadas e Não Ordenadas

Você pode criar listas ordenadas e não ordenadas usando as tags <ol> e <ul>:

Lista Ordenada

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada

  • Item A
  • Item B
  • Item C

Links e Citações

Você pode adicionar um link usando a tag <a>:

Visite o Meu Perfil DIO

Visite o Meu GitHUb

Visite o Meu Linkedin

Para citar alguém, você pode usar a tag <blockquote>:

"Esta é uma citação de exemplo usando a tag <blockquote>."

Subscrito e Sobrescrito

Use <sub> para texto subscrito e <sup> para texto sobrescrito:

Fórmula química da água: H2O

Valor ao quadrado: X2
